## Phase 0 – Python Foundation

### Core Python

- Variables · Data Types · Input/Output · Operators
- Conditionals · Loops · Functions · Modules · Exception Handling

### Python Collections

- List · Tuple · Dictionary · Set · String

### Python Features

- List / Dict / Set Comprehension
- `enumerate()` · `zip()` · `sorted()` · `lambda`
- `map()` · `filter()` · `any()` · `all()`
- `Counter` · `defaultdict` · `deque` · `heapq` (basic) · `itertools`

---

## Phase 1 – Complexity

**Understand:** Time Complexity · Space Complexity · Big O · Big Omega · Big Theta

**Recognize:** O(1) · O(log n) · O(n) · O(n log n) · O(n²)

---

## Phase 2 – Arrays & Strings

**Master:** Traversal · Insert/Delete · Reverse · Rotate · Prefix/Suffix · Frequency Counting

**Target:** 40–50 problems

---

## Phase 3 – Two Pointers

**Patterns:** Opposite Direction · Same Direction · Fast & Slow (Array) · Merge · Partition

**Target:** 25–30 problems

---

## Phase 4 – Sliding Window

### Fixed Window

- Sum · Average · Max · Min

### Variable Window — Positive Numbers

- Longest · Shortest

### Frequency Map Window

- Distinct Characters · Character Count · At Most K · Exactly K · Minimum Window

**Target:** 40+ problems

---

## Phase 5 – HashMap / Set

**Patterns:** Frequency Count · Lookup · Deduplication · Pair Finding · Grouping

**Key Problems:** Two Sum · Group Anagrams · Top K Frequent · Happy Number · Longest Consecutive Sequence · Contains Duplicate

**Target:** 30+ problems

---

## Phase 6 – Prefix Sum

**Master:** 1D Prefix Sum · Running Sum · Difference Array · HashMap + Prefix Sum

**Key Problems:** Running Sum · Subarray Sum Equals K · Continuous Subarray Sum · Range Sum Query

**Target:** 25+ problems

---

## Phase 7 – Binary Search

**Patterns:** Classic · Answer Search · Binary Search on Value · Binary Search on Array

**Key Problems:** Search Insert · First Bad Version · Koko Eating Bananas · Capacity to Ship Packages · Find Peak Element

**Target:** 30+ problems

---

## Phase 8 – Sorting

**Algorithms:** Python Sort · TimSort · Merge Sort · Quick Sort (Concept) · Counting Sort · Bucket Sort (Concept)

**Target:** 20+ problems

---

## Phase 9 – Matrix

**Patterns:** Traversal · Simulation · Boundary · Layer

**Key Problems:** Spiral Matrix · Rotate Image · Set Matrix Zeroes · Search Matrix

**Target:** 20+ problems

---

## Phase 10 – Intervals

**Patterns:** Merge · Overlap · Insert · Meeting Rooms

**Target:** 20+ problems

---

## Phase 11 – Stack

**Patterns:** Balanced Parentheses · Monotonic Stack · Expression

**Key Problems:** Valid Parentheses · Daily Temperatures · Next Greater Element · Min Stack

**Target:** 25+ problems

---

## Phase 12 – Queue / Deque

**Patterns:** Normal Queue · Deque · Circular Queue · Sliding Maximum

**Target:** 15+ problems

---

## Phase 13 – Greedy

**Patterns:** Local Optimum · Sorting + Greedy · Interval Greedy

**Key Problems:** Jump Game · Gas Station · Assign Cookies · Lemonade Change

**Target:** 25+ problems

---

## Phase 14 – Recursion

**Understand:** Call Stack · Backtracking Basics · Tree Recursion (Concept Only)

> No Tree problems — concept understanding only.

---

## Phase 15 – SQL _(Most Important — run in parallel throughout)_

**Master:** Joins · Aggregations · Window Functions · Ranking · CTEs · Recursive CTE · Pivot · Unpivot · Date Functions · Analytical Queries

**Target:** 100+ problems

---

## Phase 16 – Pandas

**Master:** DataFrame · GroupBy · Merge · Join · Pivot · Apply · Vectorization · Missing Values · Datetime

**Target:** 100+ problems

---

## Phase 17 – PySpark

**Master:** RDD (Concept) · DataFrame · Transformations · Actions · Window Functions · Partitioning · Caching · Broadcast · Skew Handling · Optimization · Spark SQL

## Phase 18 – Data Engineering Algorithms

_Instead of Trees and Graphs, focus on:_

- Merge Logs · Deduplication · Streaming · Chunk Processing
- Pagination · Batch Processing · Sliding Windows · Rolling Aggregations
- Reservoir Sampling (Concept) · Bloom Filter (Concept) · HyperLogLog (Concept)

## Phase 19 – System Design (Data)

- Python coding rounds
- SQL scenario questions
- PySpark optimization
- Data Engineering scenario & design questions

## What to Skip (for Data Engineer)

> Unless you later target Software Engineer roles — these are **not high ROI** for DE interviews.

- ❌ Linked Lists
- ❌ Trees / BST
- ❌ Graphs
- ❌ Trie
- ❌ Segment Tree / Fenwick Tree
- ❌ Union Find
- ❌ Dynamic Programming (deep coverage)
- ❌ Advanced Backtracking
